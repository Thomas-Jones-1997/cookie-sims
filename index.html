How to install

Step One

Find a build that you like to download, go to overview page and download.

Step Two

Once the file has downloaded to your PC/Mac find the location of the folder and extract files.

Step Three

Copy the folder contents to Documents > Electronic Arts > The Sims 4 > Tray

Step Four

The fun step providing you've put the files in the correct folder you can fire up the game and have fun
